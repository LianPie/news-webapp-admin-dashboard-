# News Website

A news web application built with ASP.NET MVC.

## Features
- Admin panel for managing news articles
- Create, edit, and delete articles
- Manage categories and tags

## Technologies
- ASP.NET MVC
- C#
- Entity Framework
- SQL Server
- HTML, CSS, JavaScript

## How to Run
1. Clone the repo
2. Open in Visual Studio
3. Update connection string in `Web.config`
4. Run the SQL script in `/Database` folder to create the database
5. Press F5 to run

## What I Learned
- MVC architecture
- Manual database design with SQL Server
- CRUD operations with Entity Framework (database-first approach)
- Building an admin panel from scratch
